About This Application

The Hyper Allocation Portal is a full-featured, role-based internal web application designed specifically for Adira Telecom's Realme distribution operations in Gujarat and beyond.

It enables seamless coordination between Admin teams (upload allocations & manage data), RSDs (oversee territory), Sales Managers (approve adjustments), and Area Distributors (confirm purchase orders & submit payments).

All data is secured via Google Firebase Realtime Database with role-specific access, PBKDF2 password hashing, and path-level security rules.

Key Features

Role-Based Access Control
4 distinct roles with completely isolated views & data access
Master Updation Panel
Upload Allocations, Prices & Balances via Excel in one place
RSD Territory Management
AD overview, allocation reports & broadcast announcements
Finalized PO Export
Pivot-format Excel matching exact business PO template
Payment Verification
UTR-based payment tracking & admin bulk-verify multi-select
Secure Authentication
PBKDF2 hashed passwords with per-user salt via Web Crypto API
Real-time Firebase Sync
Live data updates across all connected users instantly
Mobile Responsive
Slide-in sidebar drawer, stacking layouts & touch-friendly UI

Technology Stack

Frontend: HTML5, Vanilla CSS3, Vanilla JavaScript (ES6+)
Database: Google Firebase Realtime Database
Authentication: Custom PBKDF2 Password Hashing (Web Crypto API)
Excel Processing: SheetJS (xlsx.js) — read & export .xlsx files
Charts: Chart.js — bar & doughnut real-time visualisations
Icons: Lucide Icons — consistent open-source icon system
Fonts: Plus Jakarta Sans (Google Fonts)
